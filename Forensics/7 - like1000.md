# Solución
Al descargar nuestro archivo, vemos que es un archivo comprimido sobre otro archivo comprimido (básicamente del 1000 al 1) por lo que se aplicará un script como el siguiente:
```bash
for i in {1000..1}; do tar -xvf "$i.tar"; done
open flag.png
```
